# Country Comparison Application

Vercel Link: https://country-comparison-gn4opqsq9-dalton-willis.vercel.app/

This Country Comparison App uses Node.js, Express, and the REST Countries API to allows users to search for countries, compare country information, visualize population data, view country locations on a map, and save countries to a Supabase database. 

The main functionality includes:

- Searching for countries by name
- Viewing country details such as capital, region, population, area, languages, currencies, and borders
- Comparing selected countries side by side
- Viewing a population chart using Chart.js
- Viewing selected countries on a Leaflet map
- Saving countries to a Supabase database
- Loading saved countries from the database
- Adding saved countries back into the comparison dashboard

## Target Browsers

This application is designed for contemporary desktop browsers, including:

- Google Chrome
- Microsoft Edge
- Safari

The application uses responsive CSS and should also work on modern mobile browsers, including:

- iOS Safari
- Android Chrome

The primary target environment is desktop browsers because the comparison dashboard, chart, and map are easiest to use on larger screens.
