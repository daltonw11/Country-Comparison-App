About the Project

A database-backed country comparison dashboard.

This application helps users search for countries, compare useful indicators, and save selected countries to a Supabase database for later review.

Information Problem

Country information is often spread across different websites. Users may want a simple way to compare countries by population, area, region, capital, languages, currencies, and borders.

Purpose of the Application

The dashboard gives students, travelers, and researchers one place to search, compare, visualize, and save country information.

External Data Source

The backend retrieves live country data from the REST Countries API. The frontend does not call that external API directly.

Database

Supabase stores saved country entries. The backend includes routes for reading from and writing to the database.

Frontend Libraries

Chart.js is used for visual country comparisons, and Leaflet.js is used to display selected countries on a map.

Application Pages

The project includes a home page, an about page, and a compare page that provides the project-specific functionality.
