About

Information Problem

Many users need a simple way to compare countries across geographic, demographic, and basic economic-related indicators. Existing resources can be hard to compare side by side.

Purpose of the Application

This dashboard helps students, travelers, and researchers search for countries, compare them visually, and review country facts in one place.

Data Source

The site uses the REST Countries API to retrieve structured information such as population, region, capital, languages, currencies, borders, and time zones.

Limitations

This version focuses on broad country data and does not include deeper economic data such as GDP or inflation unless more APIs are added later.

Help
